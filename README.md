Manta Airways ✈️

Manta Airways is a modern airline booking web application built with Next.js.
This project demonstrates a responsive flight booking interface with a clean UI and scalable architecture using modern web technologies.

🚀 Tech Stack

Framework: Next.js (App Router)

Language: TypeScript

Styling: CSS / Tailwind

Font Optimization: next/font

Package Manager: npm / yarn / pnpm / bun

📦 Installation

Clone the repository
git clone https://github.com/littletangkwa/manta-airways.git
cd manta-airways

Install dependencies
npm install

Run the development server
npm run dev

🌐 Running the Project
After starting the development server, you can open the project in your browser:
Main URL
http://localhost:3000

Alternative URL (if another service is running or configured)
http://localhost:3001

The page will automatically reload when you edit files.
🛠 Development
You can start editing the main page at:
app/page.tsx

Next.js will automatically update the page as you modify the file.
This project also uses next/font to optimize and load the Geist font family.
